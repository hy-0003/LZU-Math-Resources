## 📖 项目介绍 (Introduction)

本项目旨在整合 **兰州大学数学与统计学院** 及 **萃英学院（数学班）** 本科阶段的核心课程资料

我们深知一份清晰的笔记、一套完整的往年真题对于理解晦涩的数学概念有多么重要。建立这个仓库的初衷，是希望打破届数之间的“信息壁垒”，让优质的复习资源（笔记、作业参考、考试回忆）能够在萃英山下代代相传
